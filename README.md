|---------|-------------|
| **🏠 Hero Section** | Three.js 3D canvas animations with GSAP ScrollTrigger |

## 🚀 Quick Start

1. **Clone/Download** the repository
2. **Open** `index.html` in your browser
3. **That's it!** No build tools required

```bash
# Just open in browser
open index.html
# Or drag to Chrome/Firefox
```

**Works offline** - All assets are local/CDN fallbacks.

## 📁 Directory Structure

```
Real-Estate-Website/
├── index.html              # Homepage
├── properties.html         # Property listings + filters + map
├── property-detail.html    # Single property page
├── mortgage-calculator.html # Interactive calculator
├── agents.html             # Agent profiles
├── about.html              # Company about
├── services.html           # Services offered
├── blog.html               # Blog listing
├── blog-detail.html        # Single blog post
├── contact.html            # Contact form
├── compare.html            # Property comparison
│
├── css/
│   ├── style.css           # Main styles + design tokens
│   ├── components.css      # Reusable components
│   ├── responsive.css      # Mobile breakpoints
│   ├── animations.css      # CSS animations
│   └── pages/*.css         # Page-specific styles
│
└── js/
    ├── main.js             # Core functionality (favorites, navbar, etc.)
    ├── properties.js       # Property filtering/search
    ├── mortgage.js         # Calculator logic + charts
    ├── map.js              # Leaflet map integration
    ├── animations.js       # GSAP/AOS init
    ├── counter.js          # Animated counters
    └── three-hero.js       # 3D hero canvas
```

1. **Branding**: Update colors in `css/style.css` (CSS variables)
2. **Content**: Edit HTML files directly
3. **Properties**: Add real listings in `properties.html`
4. **API Integration**: Connect to your MLS/CRM in `js/properties.js`
